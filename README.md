# 내 훈련 기록 — Personal Training Log PWA

개인 가민 훈련 기록 및 대회 예측 앱

## 배포 방법

### GitHub에 올리기
1. GitHub에서 새 저장소 생성 (이름: `garmin-log`)
2. 이 폴더의 파일들을 전부 업로드

### Vercel 배포
1. vercel.com 로그인
2. "New Project" → GitHub 저장소 선택
3. Framework: **Vite** 선택
4. Deploy 클릭

### 아이폰에 설치
1. Safari로 Vercel URL 접속
2. 하단 공유 버튼(□↑) 탭
3. "홈 화면에 추가" 탭
4. 완료 — 앱 아이콘으로 실행 가능
